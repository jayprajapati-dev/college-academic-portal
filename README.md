# Smart College Academic Portal

A web-based academic management system for colleges.

## Project Description
This project provides a centralized academic portal where
students can view semester-wise and branch-wise subject details
such as subject name, code, marks distribution and exam type.

Teachers (Admin) can post notices and assignments with deadlines
which are visible only to related students.

## User Roles
- Admin (Teacher)
- Student

## Technology Stack
- Frontend: React JS
- Backend: Node JS + Express
- Database: MongoDB

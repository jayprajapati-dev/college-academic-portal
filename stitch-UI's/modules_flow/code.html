System Modules

A technical directory of the core functional modules facilitating academic administration. This structure supports granular data management across departments with defined access protocols and system-wide integration.

shield_person

Authentication & Roles

Admin-managed module for assigning role-based permissions (Admin, HOD, Teacher, Student) with enforced access limits and session security.

group

User Management

Comprehensive profile handling, enrollment records, and directory management systems for the entire institution.

account_tree

Academic Structure

Organized mapping of departments, academic batches, and semester cycles tailored to institutional hierarchy.

grade

Subject & Marks

Management of Theory, Practical, Project, and Internship subjects. Supports Internal (PA1/PA2) and External (GTU) exam structures with admin-configurable weightage settings.

task_alt

Tasks & Submission

Modular tracking for Assignments and Tasks. Features include real-time status (Pending/Submitted/Late), submission timestamps, and integrated teacher remarks.

campaign

Notices

Centralized announcement board featuring priority tagging, real-time push notifications, and archive search.

event_available

Attendance

Teacher-led daily entry interface with automated aggregate percentage calculation and visual shortage indicators for low-attendance monitoring.

calendar_view_week

Timetable

Day-wise slot management featuring Teacher assignments and Room/Lab number allocation, restricted for modification by Admin and HOD roles.

local_library

Materials & Resources

Subject-wise repository for Syllabus, Notes, Lab Manuals, GTU Past Papers, and Model Test Papers accessible by students and faculty.

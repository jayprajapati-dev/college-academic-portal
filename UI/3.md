Design a high-end, professional student registration page for an academic portal.

====================
STYLE
====================
- Clean, modern, premium
- Card-based layout
- Soft shadows and rounded corners
- Official academic feel

====================
LAYOUT
====================
Split layout:

LEFT SECTION:
- Headline:
  “Create your SmartAcademics account”
- Short points:
  - View academic details
  - Access materials
  - Track assignments

RIGHT SECTION:
- Registration form card

====================
FORM FIELDS
====================
- Full Name
- Enrollment Number
- Mobile Number
- Password
- Security Question (fixed):
  “What is your favourite color?”
- Answer input (case-insensitive)

====================
BUTTONS
====================
- Primary “Register” button
- Text link: Already have an account? Login

====================
EXTRA DETAILS
====================
- Guided spacing (step-like feel)
- Clean validation states
- Fully responsive

Design must look official and trustworthy.

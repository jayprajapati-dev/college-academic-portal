## üîç Troubleshooting

### Admin cannot login
- Check MongoDB is running
- Verify seed data was created
- Check .env JWT_SECRET is set

### Student registration fails
- Check email is unique
- Check password is 6+ characters
- Check backend is running

### First login not triggering
- Check passwordChangeRequired = true
- Check user.status = pending_first_login
- Check token is valid

### Forgot password fails
- Check security question is set
- Check security answer is hashed
- Verify case-insensitive flag

### Dashboards not loading
- Check user role is correct
- Check token is valid
- Check API endpoints are working
- Check CORS is configured

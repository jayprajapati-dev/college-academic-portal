# Flowchart Prompt 06 - Attendance Marking

Goal: Show attendance marking and student view.

Prompt:
Create a flowchart for attendance. Steps: Teacher selects subject -> Select date and session type (lecture/lab) -> Load student list -> Mark present/absent -> Save attendance -> Student views attendance summary. Use 7 to 9 nodes, top-to-bottom layout.

Must include nodes:
- Select Subject
- Select Date and Session
- Load Student List
- Mark Attendance
- Save Attendance
- Student Views Summary

Notes:
- Keep it compact.

# Flowchart Prompt 01 - System Overview

Goal: Create a clean system overview flowchart for the Smart College Academic Portal.

Prompt:
Create a professional system overview flowchart for the Smart College Academic Portal. Show four roles (Admin, HOD, Teacher, Student) connecting to their dashboards. From dashboards, show access to core modules: Subjects, Tasks, Notices, Attendance, Timetable, Library, Notifications. Show a single shared backend (Node/Express API) and database (MongoDB) behind the modules. Use a left-to-right layout, 8 to 12 nodes, and simple verbs (Login, Access, View, Manage). Keep labels clear and formal.

Must include nodes:
- Portal / Login
- Role Selection (Admin, HOD, Teacher, Student)
- Role Dashboards (can be a single node: Role Dashboards)
- Core Modules (Subjects, Tasks, Notices, Attendance, Timetable, Library, Notifications)
- Backend API
- Database (MongoDB)

Notes:
- Do not add external services.
- Keep to one page.

# Flowchart Prompt 05 - Notice Publishing

Goal: Show notice creation, targeting, and student viewing.

Prompt:
Create a flowchart for notice publishing. Steps: Admin/HOD/Teacher creates notice -> Select target roles/branch/semester -> Save notice -> Notification generated -> Students open notice board -> Notice displayed -> Notice archived after expiry date. Use 7 to 10 nodes. Use a top-to-bottom flow.

Must include nodes:
- Create Notice
- Select Target Audience
- Save Notice
- Generate Notification
- Student Opens Notice Board
- Notice Displayed
- Notice Archived (after expiry)

Notes:
- Keep wording simple and professional.

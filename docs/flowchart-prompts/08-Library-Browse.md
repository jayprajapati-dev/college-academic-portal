# Flowchart Prompt 08 - Library Browsing

Goal: Show subject-wise library browsing.

Prompt:
Create a flowchart for library browsing. Steps: Student opens library -> Select subject -> Fetch books -> Display list -> Open book details -> Optional search or filter by author. Use 6 to 8 nodes, top-to-bottom layout.

Must include nodes:
- Open Library
- Select Subject
- Fetch Books
- Display Book List
- View Book Details
- Search/Filter (optional)

Notes:
- Keep labels simple.

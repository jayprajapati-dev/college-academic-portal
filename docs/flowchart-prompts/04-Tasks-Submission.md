# Flowchart Prompt 04 - Task Creation and Submission

Goal: Show end-to-end task creation and student submission.

Prompt:
Create a flowchart for the task lifecycle. Steps: Teacher creates task -> Enter title, description, due date -> Save task -> Task appears in student list -> Student opens task -> Decision: before due date? -> Student submits response/file -> System stores submission -> Teacher reviews -> Teacher updates status -> Student sees updated status. Use 9 to 12 nodes with one decision node for due date (On time vs Overdue). Use left-to-right layout.

Must include nodes:
- Teacher Creates Task
- Enter Task Details
- Save Task
- Task Visible to Students
- Student Opens Task
- Due Date Check (decision)
- Submit Response/File
- Store Submission
- Teacher Reviews
- Update Status
- Student Sees Status

Notes:
- Keep labels short and formal.
